Informasi Penting

Tentang Iklan:

Beberapa video mungkin mengandung iklan dari penyedia layanan streaming pihak ketiga. Kami tidak dapat mengontrol iklan tersebut sepenuhnya, jadi mohon maklum ya. Iklan membantu penyedia layanan untuk tetap menyediakan konten gratis.

Tentang Kecepatan Loading:

Website ini masih baru dan dalam tahap pengembangan. Data anime kadang keluar agak lama karena server masih terbatas. Mohon bersabar ya saat loading, dan terima kasih atas pengertiannya! 🙏

Kekurangan & Batasan

Website ini masih dalam tahap pengembangan, jadi mungkin ada beberapa hal yang belum sempurna:

  • Koleksi anime masih terbatas dan terus ditambah
  • Beberapa fitur mungkin masih dalam pengembangan
  • Server kadang lambat saat jam sibuk
  • Subtitle mungkin belum tersedia untuk semua anime
  • Ada beberapa bug yang sedang diperbaiki

Mohon maklum ya, dan terima kasih atas pengertiannya! 🙏

Request Anime

Ada anime yang pengen kamu tonton tapi belum tersedia? Tenang, kamu bisa request anime favorit kamu!

Kalau mau request anime, bisa langsung lewat Saweria dengan kasih pesan anime apa yang kamu mau. Nanti admin akan usahakan untuk upload anime yang kamu request.

Request bersifat sukarela ya, tidak ada paksaan sama sekali. Kalau support via Saweria, itu sangat membantu untuk maintenance website dan upload konten baru.
